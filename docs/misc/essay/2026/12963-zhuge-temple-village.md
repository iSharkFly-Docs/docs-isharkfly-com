# 逐步被交通边缘和空心化的 —— 诸葛庙村

> 🔔 原文：https://www.isharkfly.com/t/topic/12963


诸葛庙村与百家渡位于湖南省永州市零陵区朝阳街道，是一个历史悠久、人文底蕴深厚的古村落与古渡口。

相传三国时期，诸葛亮率兵南征驻扎于此（零陵南郊潇水西岸），后人为了纪念他而建立诸葛庙，村庄因此得名诸葛庙村。

因其深厚的崇文之风，历史上孕育了众多文人名士，且地理位置优越，邻近柳子祠和柳宗元《永州八记》中的多处遗址（如石涧遗址）。

该村历史悠久，但在20世纪70年代，村内庙宇的主体被拆除。

建安十三年（公元208年），一支玄甲长矛军纪井然的队伍快马加鞭从零陵郡东南方向直逼零陵城。坐在队伍中马车上的就是刘皇叔——刘备，在他身边的是军师诸葛亮。原来，赤壁大战之后，刘备采纳马良南征武陵、长沙、桂阳、零陵四郡的建议，挥师南下，亲自带着一万五千将士攻打零陵来了（见《三国演义》第五十二回）。

为了慎重起见，军师诸葛亮派张飞为先锋，自己和刘皇叔居中，赵云殿后，呈一字形行进，从东南方向迂回扑向零陵。

零陵太守刘度之子刘贤与上将邢道荣率兵数千出城三十里拒敌，想把战火拒之城外。谁知双方一交锋，邢道荣就不敌张飞、赵云之勇，败退回城。

诸葛亮挥兵追至潇水边，但见过河舟桥已拆，遂在南岸安营扎寨。不久，诸葛亮略施小计，派张飞先重迂回到零陵城北二十里外（今接履桥一带）待命，让对方麻痹大意。

一天晚上，粗中有细的张飞带着十多个扮成渔民的士兵，悄悄来到西城墙脚下，沿着通往潇水河里的一条水沟爬进城里，与赵云里应外合，攻入了零陵城。

等诸葛亮等人走后，人们感于诸葛亮的智慧和亲和，就在他曾经屯兵的地方建了一座庙宇来纪念他，叫做诸葛庙。

后来，这一带集聚了不少人，并形成了一个较大的村子，叫诸葛庙村。

元和七年（公元812年），在永州谪居了七年之久的柳宗元，继续以游山玩水写作方式来驱除内心的孤独与寂寞。这年秋季的某日午后，柳宗元与堂弟柳宗直、表弟卢遵等人从愚溪附近出发，走水路，经朝阳岩向东南到芜江，发现风景较好的有三处，而风景最好的就是袁家渴。由于天色已晚，大家虽然意犹未尽，但只好乘舟归去。

回去之后，柳宗元凭印象写下了《袁家渴记》。过了一段时间，永州刺史韦宙闲着无事，来找柳宗元闲逛。

柳宗元想起上次去过的袁家渴，就领着韦宙等人去那里游览，“又折西行，旁陷岩石下，北坠小潭”，发现了石渠。石渠之美，让刺史韦宙十分高兴，他怂恿柳宗元继续走一走。于是，大家从石渠的桥上向西北走，翻过一座土山，沿北坡走了大约一里路，发现老百姓又在溪水上架了一座桥，桥边有一小村，就是今天的杨家。

那溪流比石渠的水量大三倍，石头有的像床，有的像门堂的基石像筵席上摆满菜肴，有的像用门槛隔开的内外屋，水流像纺织物的花纹，水泉咚响声像是弹琴声。

回去之后，他写了两篇游记《石渠记》和《石涧记》，丰盈了流传千古的《永州八记》。

百家渡位于诸葛庙村下方，旧名百家渡，也是诸葛庙渡，曾是零陵至道县的陆水陆交通必经之地。

文豪在此留下诗作，最著名的是苏轼的《百家渡》（“百家渡西日欲落，青山上下猿鸟乐”），以及杨万里的《过百家渡四绝句》。

百家渡遗址和石涧遗址已被列为区域级文物保护单位，但因年代久远损毁较为严重。

![2026-04-22_12-00-07|690x488](upload://q2blusgER9aTNqIlxu52CsxNi4B.jpeg)

截至2025年，诸葛庙村面积 2.12 平方千米，人口 338 户 821 人。

诸葛庙村濒临潇水，在南津渡大桥建成前，主要依靠 “百家渡” 渡口码头出行。

![2026-04-22_12-00-53|690x462](upload://eYztBWUHN6ZgYQu476VWldhVt0y.jpeg)

因道路和交通条件的改变，诸葛庙村人们的出行方式又发生了根本性的变化。

在南津渡大桥还没有通车之前，诸葛庙村的村民的主要通勤目的地对象为零陵城内。因此百家渡是诸葛庙及附近村民进入零陵城区的主要选择。

渡口因人而兴，小时候每一天的早上都会有村民挑着自己的蔬菜进入零陵城区贩卖。

日落而息，村民又会在结束一天贩卖的以后乘船返回自己的村落，同时附近的村民的孩子，如需要进入零陵城内就读初中和高中也会通过这个渡口在学校和家庭之间进行往返。

每日都会看到类似现在公共交通早晚高峰的繁忙。

在上个世纪 90 年代的中叶，零陵南津渡大桥建成通车，往返于零陵城区和郊区的有了更多的选择。

同时随着改革开放的提速和零陵城区规划城建的整体北移，诸葛庙村被越来越边缘和空心化。

昔日的百家渡码头已经不再繁荣，随着城市城镇化建设的快速推进，人声鼎沸的百家渡也仅仅留存在人们的记忆中了。
