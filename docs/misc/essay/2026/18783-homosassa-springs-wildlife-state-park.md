# Ellie Schiller Homosassa Springs Wildlife State Park

> 🔔 原文：https://www.isharkfly.com/t/ellie-schiller-homosassa-springs-wildlife-state-park/18783


这次佛罗里达之行偶遇的宝藏公园。

在原本的计划中，没有考虑过去参观佛罗里达的公园。

网上有关这个公园的攻略也很少，基本很少有人提及，更多的来到佛罗里达考虑的都是去迪斯尼。

![2026-05-29\_09-05-53|690x469](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/3/e/3ed57e1731ba6598eb77d4d150d3ec644d7b4b72_2_690x469.jpeg)

公园的内部还饲养了海牛，如果在外面没有机会看到野生的海牛，那么还是可以通过公园看到海牛。

如果去佛罗里达是夏天的话，海牛基本上就游出去了。

![2026-05-29\_09-06-15|690x483](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/e/0/e0551120fb9d3194b54df1e325b517921e0c5c0f_2_690x483.jpeg)

参观时间留两个小时足够了，票价也很便宜，几十块钱就能看到各种佛罗里达本地的野生动物。

那以前还可以通过水路和陆路的两种方式从售票处入园。

貌似在2025年的一场飓风，动物园损失惨重，一些场馆也是处于关闭的状态。

![2026-05-29\_09-06-38|690x426](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/4/5/45658c97c2d1403bd9da74d8dc4dc27f284403c2_2_690x426.jpeg)

原本还养了一头河马，是从圣地亚哥动物园运过来的，但不幸已经离世了。

这个宝藏动物园还是很值得一去，能够感受到佛罗里达的温度。
