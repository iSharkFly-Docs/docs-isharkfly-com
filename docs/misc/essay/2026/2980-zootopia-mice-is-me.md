# 感觉疯狂动物城说的小仓鼠就是自己

> 🔔 原文：https://www.isharkfly.com/t/topic/2980

很多年前去亚马逊的时候，能感觉到活力，这次去亚马逊总感觉死气沉沉。

![2026-07-10_21-11-49|670x500](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/b/3/b33d6d3475c8adff07e8843a49fddbaa10c016c6_2_670x500.jpeg)

美国的科技大厂经过几次裁员和清洗以后，现在里面的人估计个个都是小心翼翼的。

给人的感觉是不求做对，但求不做错，所有的流程和事情都照本宣科，还是一堆的印度人，但感觉不到活力了，他们更多的就像完成一个每天都去做的工作而已，没有说把这个东西怎么做得更好。

门口还给一个稀烂的没有任何收藏价值的工牌，打印机纸介质打印的，连塑料片都不舍得给。

哪怕就算这么一个稀烂的 visitor 卡，还因为没佩戴好，被别人问了好几次，就是过来一天办点事情，还被人盯着问，你的工牌呢？

说心里话，对你们这个工牌也没有什么太大兴趣，本身也没有收藏价值。

现在这种普通 RFID ID 的普通白板卡便宜的不要能再便宜，大不了就投资一个打印机嘛，打一个漂亮的画上去不行吗？

实在不行，再不济也可以从中国淘宝或者拼多多买一点漂亮的挂绳行不行，要提高员工的认同感，细节上面上点心好不好？

就盯着别人挂没挂工牌，就挂上这破工牌是能骗两罐可乐还是骗顿免费中饭呀。

![2026-07-11_08-08-16|689x293](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/7/b/7b2ac90361bdec56787cb43b2563ce67640f1267_2_689x293.jpeg)

突然觉得很多人，包括自己和《疯狂动物城》里面这些打工的小老鼠是同样的生活。

穿着最体面的衣服，在最豪华的办公区，但是吃着最便宜的冰棍。

看了这么多次，还是第1次注意到上面的银行的名字叫做 lemming Brothers bank，这显然就是在讽刺的是雷曼兄弟银行。

片中仓鼠职员排着整齐划一的队伍转弯，人手一只冰棍，辛辣地讽刺了金融圈盲目跟风和随波逐流的投机心理。

《疯狂动物城》电影上映这么多年了，里面的彩蛋还在不停的被不经意的注意到，真的算是一部宝藏动画片。
