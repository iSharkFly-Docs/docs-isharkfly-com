# Savers 美国社区常见的 Thrift store （ 二手店）

> 🔔 原文：https://www.isharkfly.com/t/savers-thrift-store/10062

在美国，**Savers**（也称 Value Village）与 **Goodwill** 是最大的两家二手旧货（Thrift Store）零售商。

**Savers（营利性企业）：** 是一家营利性企业。

他们不直接接受公众的免费“捐赠”来归自己所有，而是与慈善机构合作，按重量向合作慈善机构购买旧货。

虽然你捐赠的物品有助于资助这些慈善机构，但你在店内购物的消费额不直接捐给慈善机构。

虽然 Savers 是营业机构，但是相对 Goodwill 来说，Savers 的服务和捐赠接纳程度更高。

不少家庭对一些过季的衣服或不合适衣服都直接捐给他们了。

逛这类的商店有点像逛咸鱼实体店，里面什么老古董老电器都有，有时候还能看到很老的黑胶唱片，能不能播放音乐那就不知道了，但是东西还是有的。

更多的人是来这类商店淘换一些衣服，因为衣服都是不是捐赠就是从慈善机构购买。

Goodwill 也是和 Savers 类似的商店，Goodwill 也接受公众的捐助，从店面的布局来看，Goodwill  显得更加杂乱。

不过两家店的整体购物体验基本差不多，大差不差。
