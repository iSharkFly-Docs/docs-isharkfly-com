# 亚特兰蒂斯号航天飞机

> 🔔 原文：https://www.isharkfly.com/t/topic/13189

亚特兰蒂斯号航天飞机（英语：STS Atlantis OV-104）是美国国家航空航天局操作的航天飞机轨道器，其名称源自亚特兰蒂斯号双桅帆船。

亚特兰蒂斯号由罗克韦尔国际公司在南加州制造，并于1985年4月运抵位于佛罗里达州东部的肯尼迪航天中心。它是第四架投入运作、倒数第二架建造完成的航天飞机。

美国退役的航天飞机主要有4架，目前均已进入博物馆对外展出，其中就包括了收藏在肯尼迪航天中心的亚特兰蒂斯号（Atlantis）。

航天飞机项目现在已经关闭，留下的几架航天飞机都在美国各大博物馆内展出。

亚特兰蒂斯号航天飞机留在了佛罗里达肯尼迪航天发射中心。

航天飞机项目更像是在那个激情澎湃的年代，代表人类挑战自身，能够站得高，看得更远，结合了当时在人类黑科技的精华。

值得惋惜的是航天飞机时代已经过去，但其疯狂的设计理念会影响好几代的人。

![2026-06-20\_23-52-31|690x497](https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/optimized/3X/f/f/ffcc38006df1231cb46e743af86396050dfd1725_2_690x497.jpeg)

在 NASA 介绍影片中说到了航天飞机项目的来源，以及在整个项目进行中遇到的一些具体问题。

项目最初的目标是希望把太空旅行能够简单到类似公交化，同时还希望火箭发动机能够重新利用，因此在项目立项之初就设计了火箭发动机回收。

也许就是这些在当时过于先进的设计给整个项目带来了很多挑战，但也不得不说就也是因为能想到更远，为目前 SpaceX 的火箭回收技术带了技术积累。
