# Yarn 相关

Yarn 是一个 JavaScript 包管理器，它是由 Facebook 开发的，作为 npm（Node Package Manager）的替代品。

Yarn 的主要目的是解决 npm 早期版本中存在的一些问题，如安装速度慢、依赖管理不一致等。

虽然 npm 在近年来也有很大改进，解决了许多早期问题，但 Yarn 仍然被许多开发者使用和喜爱。
