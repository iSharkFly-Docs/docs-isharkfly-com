# Microsoft Certified: Azure Fundamentals

Microsoft Certified: Azure Fundamentals，代号为AZ900，是Azure的初级认证考试。微软的认证也是有体系的，当完成初级认证考试后，就可以申请中级认证和高级认证，继续考试。
